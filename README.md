For Scala 3, you have the option to use `refined4s` in place of `newtype` and `refined`, along with the support for `refined4s` provided by `hedgehog-extra`.

```sbt
"io.kevinlee" %% "hedgehog-extra-refined4s" % "0.6.0"
```
